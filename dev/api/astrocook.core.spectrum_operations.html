astrocook.core.spectrum_operations.rate_doublet_candidate(spectrum: SpectrumV2, region_mask_1: ndarray, region_mask_2: ndarray, multiplet_name: str, z_test: float, debug_rating: bool = False) → float[source]#

Rates a doublet candidate using the R^2 score on the NORMALIZED FLUX profiles (Power-Law Model).

Physics: tau_1 = gamma * tau_2 ==> F_1 = F_2 ^ gamma

NOTE: region_mask_1 is the BLUER region (e.g., CIV_1548)

region_mask_2 is the REDDER region (e.g., CIV_1550)

astrocook.core.spectrum_operations.find_signal_peaks(signals_dict: Dict[str, ndarray], z_grid: ndarray, sigma: float, distance_pix: int, prominence: float | None) → List[Tuple[str, float, float]][source]#

Finds and filters peaks in the identification signals. Replicates the core logic of V1’s _systs_like.

* MODIFIED to be a generic peak finder. * * It no longer smooths or runs a ‘best-guess’ filter. *

Parameters:

  • signals_dict – Dict of {series_name: signal_array}

  • z_grid – The common redshift grid.

  • sigma – The minimum peak height (signal score).

  • distance_pix – Minimum distance between peaks in pixels (indices).

  • prominence – Minimum prominence of peaks (absolute signal value).

Returns:

List of (series_name, z_peak, score)
