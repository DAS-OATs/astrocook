class astrocook.gui.qt_workers.WorkerSignals[source]#

Bases: QObject

Defines the signals available from a running worker thread. - finished: Emits the new SessionHistory object on success. - error: Emits a tuple of (line_number, line_text, error_message). - progress: Emits a string to update the user (e.g., “Running line 5…”)
